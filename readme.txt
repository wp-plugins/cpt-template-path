Adds Custom Post Type slug as folder path in template hierarchy.

== Description ==

This plugin adds Custom Post Type slug as folder path to template hierarchy.

Like, if you a custom post type "movie". So, templates can be placed as below:
your_theme/movie/single.php
your_theme/movie/index.php

== Installation ==

1. Install CPT Template Path either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it.  You're ready to go!
